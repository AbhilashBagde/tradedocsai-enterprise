# TradeDocsAI — Enterprise

AI infrastructure for trade documentation. Automates extraction and human-verified structuring of complex trade invoices and packing lists.

## Tech stack

- Vite + React + TypeScript
- shadcn/ui + Tailwind CSS
- TanStack Query

## Getting started

```sh
git clone <YOUR_GIT_URL>
cd tradedocsai-enterprise
npm install
npm run dev
```

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start dev server on port 8080 |
| `npm run build` | Production build |
| `npm run lint` | Run ESLint |
| `npm test` | Run tests |

## Deployment

Build with `npm run build` and deploy the `dist/` folder to any static host (Vercel, Netlify, Cloudflare Pages, etc.).
